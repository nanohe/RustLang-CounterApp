  2. saturating_add / saturating_sub                                                                                                                                       
                                                                                                                                                                           
  // Before — silent overflow in release, panic in debug                                                                                                                   
  self.count += 1;                                                                                                                                                         
                                                                                                                                                                           
  // After — clamps at i32::MAX / i32::MIN instead of overflowing                                                                                                          
  self.count = self.count.saturating_add(1);                                                                                                                             
                                                                                                                                                                           
  In Rust, integer overflow is undefined behavior in C/C++ but Rust makes it explicit:                                                                                     
  - Debug builds: panic on overflow (catches bugs early)
  - Release builds: wraps silently (fast but surprising)                                                                                                                   
                                                                                                                                                                         
  saturating_add is a method on all integer primitives. It clamps at the boundary (i32::MAX = 2,147,483,647) instead of overflowing. You call it with                      
  self.count.saturating_add(1) — this is Rust's method call syntax, equivalent to i32::saturating_add(self.count, 1).
