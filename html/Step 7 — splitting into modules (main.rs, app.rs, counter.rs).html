---                                                                                                                                                                      
  Rust concepts introduced                                                                                                                                                 
                                                                                                                                                                           
  1. Declaring modules with mod                                                                                                                                            
                                                                                                                                                                           
  // src/main.rs                                                                                                                                                           
  mod app;
  mod counter;                                                                                                                                                             
                                                                                                                                                                           
  mod counter; tells Rust: "find a file at src/counter.rs and treat it as a module named counter". No separate build config needed — Cargo discovers it through main.rs.   
  The module tree always has main.rs (or lib.rs) as its root.                                                                                                              
                                                                                                                                                                           
  ---                                                                                                                                                                    
  2. pub — controlling visibility
                                                                                                                                                                           
  By default everything in Rust is private — only visible within the same module. You opt into sharing with pub:
                                                                                                                                                                           
  pub struct Counter { ... }   // visible outside counter.rs                                                                                                             
  pub fn save(...) { ... }     // visible outside counter.rs                                                                                                               
                                                                                                                                                                           
  pub struct CounterApp { ... } // visible outside app.rs                                                                                                                  
                                                                                                                                                                           
  Fields also need pub if you want other modules to read/write them directly:                                                                                              
  pub struct Counter {
      pub name: String,                                                                                                                                                    
      pub count: i32,                                                                                                                                                    
      pub category: Category,
  }                                                                                                                                                                        
   
  This is unlike most languages — Rust makes you think explicitly about your public API surface.                                                                           
                                                                                                                                                                           
  ---
  3. use paths across modules                                                                                                                                              
                                                                                                                                                                           
  // src/app.rs
  use crate::counter::{load, save, Category, Counter, SAVE_FILE};                                                                                                          
                                                                                                                                                                           
  crate refers to the root of your project (main.rs). So crate::counter::Counter means: starting from the crate root, go into the counter module, find Counter. The use    
  statement brings them into local scope so you don't have to write the full path every time.                                                                              
                                                                                                                                                                           
  ---             
  4. The resulting module tree
                                                                                                                                                                           
  crate (main.rs)
  ├── mod app       → src/app.rs                                                                                                                                           
  │   └── CounterApp                                                                                                                                                       
  └── mod counter   → src/counter.rs
      ├── Category                                                                                                                                                         
      ├── Counter                                                                                                                                                          
      ├── save()
      └── load()                                                                                                                                                           
                  
  main.rs is now just the entry point — 10 lines. Each file has a clear, single responsibility.                                                                            
   
  ---                                                                                                                                                                      
  You've now covered all 7 steps. The app has gone from a 36-line single file to a properly structured Rust project with:
                                                                                                                                                                           
  - Owned types and borrowing (String, &str, &[T])
  - Collections (Vec<T>)                                                                                                                                                   
  - Enums and pattern matching
  - Traits (Display, Debug)                                                                                                                                                
  - Error handling (Result, ?)                                                                                                                                             
  - Modules and visibility (mod, pub, use)
