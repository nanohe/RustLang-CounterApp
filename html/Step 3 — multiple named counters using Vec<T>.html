  Rust concepts introduced                                                                                                                                                 
                                                                                                                                                                           
  1. A plain struct with a constructor                                                                                                                                     
                                                                                                                                                                           
  struct Counter {                                                                                                                                                       
      name: String,                                                                                                                                                        
      count: i32,                                                                                                                                                          
  }                                                                                                                                                                        
                                                                                                                                                                           
  impl Counter {                                                                                                                                                           
      fn new(name: String) -> Self {
          Self { name, count: 0 }
      }                                                                                                                                                                    
  }
                                                                                                                                                                           
  Counter doesn't derive Default because it requires a name — so we write a manual new() instead. Self inside impl is shorthand for the type (Counter). Self { name, count:
   0 } uses field shorthand — when a variable has the same name as the field, you don't need to write name: name.
                                                                                                                                                                           
  ---                                                                                                                                                                      
  2. Vec<T> — Rust's growable list
                                                                                                                                                                           
  counters: Vec<Counter>,   // a list of Counter values, owned by CounterApp                                                                                             
                                                                                                                                                                           
  Key Vec methods used:                                                                                                                                                    
  - .push(value) — append to the end                                                                                                                                       
  - .remove(i) — remove by index, shifts elements left                                                                                                                     
  - .iter_mut() — iterate with mutable access to each element                                                                                                            
                                                                                                                                                                           
  Vec<Counter> defaults to an empty vec via #[derive(Default)].                                                                                                            
                                                                                                                                                                           
  ---                                                                                                                                                                      
  3. Iterating with iter_mut() and enumerate()                                                                                                                             
                                                                                                                                                                           
  for (i, counter) in self.counters.iter_mut().enumerate() {
                                                                                                                                                                           
  - iter_mut() gives you &mut Counter for each element — you can modify them in place                                                                                      
  - .enumerate() wraps each item as (index, item) so you know the position
  - i is used later to know which counter to remove                                                                                                                        
                                                                                                                                                                           
  ---                                                                                                                                                                      
  4. The deferred-remove pattern                                                                                                                                           
                  
  let mut to_remove: Option<usize> = None;
                                                                                                                                                                           
  for (i, counter) in self.counters.iter_mut().enumerate() {
      if ui.button("Remove").clicked() {                                                                                                                                   
          to_remove = Some(i);
      }
  }

  if let Some(i) = to_remove {
      self.counters.remove(i);
  }

  You can't call self.counters.remove(i) inside the loop because the loop already holds a mutable borrow of self.counters — Rust won't let you mutate a collection while   
  iterating it. The fix: record the index to remove, then act after the loop ends. Option<usize> is perfect for "maybe an index" — None means nothing to remove, Some(i)
  means remove index i.
